# **App Name**:Grid Saga

## Core Features:

- Core Merge Mechanic: Classic 2048 grid merging functionality.
- Varied Grid Sizes: Choice of multiple grid sizes for different levels of complexity.
- Tile themes: Unique tile designs and themes.
- Level Progression: Level based progression, offering more complex modes, tile types and grid sizes
- Adaptive Gameplay: The game includes a tool that uses generative AI to adapt to the player's skill, dynamically suggesting puzzle variations and hints.

## Style Guidelines:

- Primary color: A vibrant purple (#9D4EDD), evoking creativity and playfulness.
- Background color: A very light purple (#F5F0FF), complementing the primary color and ensuring readability.
- Accent color: A bright pink (#EEA47F), creating a striking contrast to highlight interactive elements.
- Body and headline font: 'Space Grotesk', a contemporary geometric sans-serif font offering both a modern aesthetic and readability.
- Fluid animations for tile merging and transitions to visually reward player actions.
- Clean, minimalistic design focused on the central grid, providing an intuitive and uncluttered user experience.
- Simple and recognizable icons for features.
